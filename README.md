# Wood Foundations

The Wood Foundations mod adds five (5) new wood foundation materials in a variety of styles.

## Features

### Five (5) new foundation materials
  - __Wood (Pine):__ Unlocked automatically when foundations are unlocked
  - __Redwood__, __Walnut__, __Light__, and __Dark:__ Purchasable in the Awesome Shop for 2 Tickets each
  - Build Cost: All wood foundation varieties __require 2 Wood per foundation__ to build.
  - Usable on all 42 vanilla foundation shapes

### Complete Vanilla Material Customizer Functionality
  - Each Wood Material, once unlocked, can be applied with the customizer like any other foundation material (i.e. concrete, grip metal, etc), or they can be built directly as normal via the build gun menu.
  - Further, you can use the customizer to turn wood foundations back into any vanilla foundation.
  - Buying foundation sets from the Awesome shop (i.e. double ramps, corner ramps, etc) will also unlock the relevant wood versions for any wood varieties you have unlocked (i.e. same beahvior as default customizer materials like Concrete and Asphalt). 
  - __TL;DR: The wood materials work exactly like the vanilla foundation materials__

### Patterns and Colors
  - All wood varieties support Customizer Foundation Patterns
  - Color swatches: I've setup the textures so that the Primary color has a 30% weighted affect on the albedo of the wood, but won't 'paint' the wood that color. The secondary color applies to patterns as usual.

### Performance
  - I've taken great care to ensure these Wood Foundations are constructed the exact same way vanilla objects are built. All textures are implemented as Material Instances of base-game materials to prevent unnecessary shader overhead, and all buildables fully utilize the game's native Abstract Instance system.
